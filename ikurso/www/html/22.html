3.3. Vocabulaire

Temps moyen estimé : 3 minutes

Voici maintenant un peu de vocabulaire.

Noms

horo : heure

jaro[1] : année

mateno : matin

minuto : minute

semajno : semaine

nokto : nuit

tago : jour

vespero : soir, soirée

Verbes

atendi : attendre

demandi [2] : demander

fumi : fumer

kuri : courir

peti [3] : demander

promeni : se promener

respondi : répondre

soifi [4] : avoir soif

vivi : vivre

Adjectifs

blanka : blanc

blua : bleu

bruna : brun (marron)

flava : jaune

griza : gris

nigra : noir

ruĝa : rouge

verda : vert

Adverbe

nur : seulement

[1] Rappelez-vous que le j est prononcé comme le y dans yoghourt.

[2] demandi : poser une question.

[3] peti : demander pour obtenir quelque chose.

[4] soifi : bien prononcer le o et le i séparément (so-i-fi).
